## Appliance Repairs Landing Page

Responsive single-page marketing site for a Johannesburg-based appliance repair service, built with HTML, CSS, and vanilla JavaScript.

## Overview

This project is a static landing page for a local appliance repair business operating in the greater Johannesburg area. It showcases services (repairs, maintenance, and cleaning for major household appliances), customer testimonials, FAQs, and clear contact options. The page is fully responsive and includes a prominent “Book Appointment” call-to-action that lets visitors quickly call or WhatsApp the business.

## Features

- **Modern responsive layout**: Works on desktop, tablet, and mobile using CSS grid and flexbox.
- **Hero section**: Bold branding, key services, service areas, and a visual appliances image.
- **Service listing**: Detailed list of appliances covered (washing machines, stoves, fridges, air conditioners, etc.).
- **“Why Choose Us” section**: Highlights door-step service, same-day availability, competitive pricing, and quality guarantees.
- **About section**: Describes business experience, value proposition, and core strengths.
- **Testimonials**: Customer reviews with ratings to build trust.
- **FAQ accordion**: Expandable Q&A about service areas, brands, pricing, and warranties.
- **Contact section**: Phone, WhatsApp, business hours, and service area info.
- **Booking modal**: Floating “Book Appointment” button that opens a modal with quick actions to call or send a WhatsApp message.
- **Smooth scrolling & UI polish**: Animated scroll behavior, subtle hover states, and navbar styling that responds to scroll.

## Tech Stack

- **HTML5**: Semantic structure for all sections.
- **CSS3**: Custom styling, layout, animations, and responsive design (no frameworks).
- **Vanilla JavaScript**:
  - Booking modal open/close logic.
  - FAQ toggle/accordion behavior.
  - Smooth scrolling for in-page navigation.
  - Navbar style change on scroll.
  - Fallback logic for the hero appliances image.
